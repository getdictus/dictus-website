  <name>Task 1: Run Lighthouse audit on /fr and /en, fix issues until 90+ on all metrics</name>
  <files></files>
  <action>
1. **Start the production server:**
   - Run `npm run build` then `npx next start` in background (port 3000)
   - Wait for server to be ready

2. **Run Lighthouse audit on /fr:**
   - `npx lighthouse http://localhost:3000/fr --output=json --output=html --output-path=./lighthouse-fr --chrome-flags="--headless --no-sandbox" --quiet`
   - Parse JSON output to extract scores for Performance, Accessibility, Best Practices, SEO
   - Log all scores

3. **Run Lighthouse audit on /en:**
   - `npx lighthouse http://localhost:3000/en --output=json --output=html --output-path=./lighthouse-en --chrome-flags="--headless --no-sandbox" --quiet`
   - Parse JSON output to extract scores for Performance, Accessibility, Best Practices, SEO
   - Log all scores

4. **If any score < 90:** Read the Lighthouse report details, identify failing audits, and fix them:
   - Performance: Check for render-blocking resources, LCP issues, CLS, unused JS
   - Accessibility: Check for missing alt text, contrast issues, ARIA problems, heading order
   - Best Practices: Check for console errors, deprecated APIs, HTTPS issues
   - SEO: Check for missing meta tags, robots issues, crawlable links
   - Apply fixes to the relevant files and re-run Lighthouse until all scores >= 90

5. **Clean up:** Remove lighthouse-fr.* and lighthouse-en.* report files. Stop the background server.
  </action>
