   - For now, render children directly without NextIntlClientProvider (added in Plan 02)

4. Create `src/app/[locale]/page.tsx`:
   - Minimal placeholder page with a centered "dictus" wordmark styled in DM Sans 200, letter-spacing -0.04em
   - Dark background demonstrating the design tokens work
   - Text "Foundation" in white-40 below the wordmark to confirm token rendering

5. Create `src/app/[locale]/not-found.tsx`:
   - Simple 404 page with "Page not found" text using brand styling

IMPORTANT: DM Sans is a variable font (do NOT specify weights). DM Mono is NOT variable (MUST specify weight: ['300', '400']).
