# Requirements: dictus Landing Page

**Defined:** 2026-04-09
**Core Value:** Convaincre un visiteur en quelques secondes que dictus est l'alternative privacy-first a la dictation vocale, et lui donner envie de tester l'app.

## v1.3 Requirements

Requirements for milestone v1.3 Donate & Desktop. Each maps to roadmap phases.

### Donation Page

- [ ] **DON-01**: Page `/donate` bilingue FR/EN accessible depuis le site, avec deux cartes cote a cote (Fiat et Bitcoin)
- [ ] **DON-02**: Carte Fiat affiche des montants suggeres (5/10/25/50 EUR) + montant libre, redirige vers Stripe Payment Link
- [ ] **DON-03**: Carte Bitcoin ouvre une invoice BTCPay Server (Pay Button ou lien direct)
- [ ] **DON-04**: Design coherent avec le design system Dictus (dark, glassmorphism, accent blue, hover glow)
- [ ] **DON-05**: Label "Contribution" (pas "Don") — PIVI Solutions est une entreprise, pas une association

### Payment Infrastructure (guides/documentation)

- [ ] **PAY-01**: Guide de setup Stripe pour PIVI Solutions (compte, Payment Link avec montant personnalisable)
- [ ] **PAY-02**: Guide de setup BTCPay Server (option recommandee avec couts, pruned node, phoenixd pour Lightning)

### Webhook Notifications

- [ ] **HOOK-01**: API route `/api/webhooks/stripe` verifie la signature Stripe et envoie un message Telegram a chaque paiement
- [ ] **HOOK-02**: API route `/api/webhooks/btcpay` verifie la signature HMAC BTCPay et envoie un message Telegram a chaque paiement
- [ ] **HOOK-03**: Messages Telegram formattes avec methode, montant, devise et date
- [ ] **HOOK-04**: Variables d'environnement documentees (TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID, STRIPE_WEBHOOK_SECRET, BTCPAY_WEBHOOK_SECRET)

### Desktop Launch Preparation

- [ ] **DESK-01**: Section telechargement avec tabs par OS (macOS / Windows / Linux) et auto-detection du systeme via navigator.userAgent
- [ ] **DESK-02**: Badges Coming Soon desactives pour chaque plateforme avec icones OS (Apple, Windows, Linux)
- [ ] **DESK-03**: Configuration centralisee des liens de telechargement dans un fichier unique (downloads.ts) pret a activer
- [ ] **DESK-04**: Mise a jour des textes du site : Dictus n'est plus mobile-only, mentionner Desktop (Mac, Windows, Linux)

### Social & Footer

- [ ] **SOC-01**: Icone X/Twitter dans le footer pointant vers https://x.com/getdictus (entre GitHub et Telegram)
- [ ] **SOC-02**: Label i18n twitter_label dans les fichiers de traduction FR et EN

## Out of Scope

| Feature | Reason |
|---------|--------|
| Dons recurrents | Complexite Stripe Subscriptions, pas necessaire au lancement |
| Recu fiscal / tax receipt | PIVI Solutions est une entreprise, pas une asso d'utilite publique |
| Stripe Checkout Sessions (backend) | Payment Links suffisent, zero backend |
| CDN blob.getdictus.com | Pas urgent tant que Desktop n'est pas public (issue #2) |
| Liens de telechargement actifs Desktop | App pas encore prete, Coming Soon d'abord |
| Nettoyage/simplification des sections | A evaluer plus tard, pas necessaire pour cette milestone |
