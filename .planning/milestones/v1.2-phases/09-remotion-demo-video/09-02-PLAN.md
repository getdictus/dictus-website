1. Create `video/src/components/Waveform.tsx`:
   - Props: `{ state: DemoState; stateStartFrame: number; width: number; height: number }`
   - Use `useCurrentFrame()` and `useVideoConfig()` from remotion
   - Convert state to phase via `DEMO_STATE_TO_PHASE[state]`
   - For each of the 30 bars:
     a. Call `computePhaseEnergy(i, phase, frame, stateStartFrame, fps)` for target energy
     b. Apply smoothing: since Remotion renders each frame independently, compute a rolling average by looking back ~5 frames. For simplicity, use the raw energy from computePhaseEnergy (the function already handles deterministic smooth behavior internally).
     c. Compute bar height: `MIN_BAR_HEIGHT + energy * (MAX_HEIGHT - MIN_BAR_HEIGHT)`
     d. Call `resolveBarColor(i)` to determine if gradient or edge
     e. Draw bar as rounded rect on canvas
   - Use canvas ref callback pattern (Remotion renders each frame, so the ref callback redraws):
     ```tsx
     <canvas ref={(canvas) => { if (!canvas) return; drawAllBars(canvas, frame, ...); }} width={width} height={height} />
     ```
   - For gradient bars: create `ctx.createLinearGradient` using WAVEFORM_COLORS.gradientStart (#6BA3FF) and WAVEFORM_COLORS.gradientEnd (#2563EB)
   - For edge bars: use `rgba(${WAVEFORM_COLORS.edgeRgb}, ${opacity})` where opacity comes from resolveBarColor
   - Bar corner radius = barWidth / 2 (fully rounded ends)
   - Center bars vertically: `y = (height - barHeight) / 2`

2. Create `video/src/components/StateIndicator.tsx`:

   - Pill-shaped container with semi-transparent dark background (rgba(0,0,0,0.4)), border-radius 999px, padding 8px 16px
   - Left side: colored dot (8px circle) using STATE_COLORS[state]
   - Right side: label text in DM Sans 300, white70
   - When state is "recording" or "transcribing": dot has a glow effect (box-shadow with state color at 50% opacity)
   - When state is "idle": hide the indicator (opacity 0)
   - Use `spring()` from remotion for opacity transitions
