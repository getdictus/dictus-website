<objective>
Migrate the Dictus landing page color system from hardcoded dark-only tokens to a CSS variable swap pattern that supports both light and dark themes, and wire up next-themes for persistence and FOWT prevention.

Purpose: This is the foundation for the entire theme system. Without the CSS variable indirection in @theme, no component color will swap. Without ThemeProvider, there is no mechanism to toggle or persist themes.

Output: globals.css with :root/:dark variable definitions and @theme indirection, ThemeProvider client wrapper, layout.tsx with provider and suppressHydrationWarning.
</objective>
