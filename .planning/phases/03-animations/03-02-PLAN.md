**src/hooks/useAnimationFrame.ts:**
- Custom hook: `useAnimationFrame(callback: (time: number) => void, isActive: boolean)`
- Uses `useRef` for requestRef and previousTimeRef
- Calls `requestAnimationFrame` in a loop when `isActive` is true
- Cleans up with `cancelAnimationFrame` in useEffect return
- Wrap callback in useCallback to avoid re-registering rAF on every render

**src/hooks/useHeroDemoState.ts:**
- Orchestrates the demo sequence shared between StateIndicator and TextReveal
- Accepts `sentences: string[]` (from translations) and `shouldReduce: boolean`
- Manages a state machine cycling through 5 states with timings:
  - `idle`: ~1.5s (dot white/30%, no text action)
  - `recording`: ~2s (dot red pulse, text words start appearing)
  - `transcribing`: ~2.5s (dot blue pulse, words continue flowing)
  - `smart`: ~2s (dot purple pulse, brief pause in text)
  - `inserted`: ~2s (dot green static, sentence complete, hold then reset)
- Total cycle: ~10s, then advances to next sentence and restarts
- Exposes: `{ currentState, stateColor, shouldPulse, stateLabel, visibleText, showCursor, sentenceIndex }`
- `visibleText`: builds word-by-word during recording+transcribing states, full text during smart+inserted
- Word timing: ~250ms base with +/- 50ms random variation for organic feel
- `showCursor`: true during active typing states (recording, transcribing, smart), false during idle and inserted
- When `shouldReduce` is true: show full sentence immediately, cycle states without pulse, no word animation
- Clean up all timeouts/intervals in useEffect return

**src/components/Hero/Waveform.tsx** ("use client"):
- Canvas element rendering 2-3 overlapping sinusoidal waves
- Import `useAnimationFrame` and `useReducedMotion` from motion/react
- Canvas fills its container (w-full, absolute positioning behind hero content)
- Canvas height: ~300px on desktop, ~200px on mobile (responsive via CSS + ResizeObserver)
- Handle retina: set `canvas.width = clientWidth * devicePixelRatio`, `canvas.height = clientHeight * devicePixelRatio`, then `ctx.scale(dpr, dpr)`
- Use ResizeObserver to update dimensions on resize
- Draw function paints 3 sine waves per frame:
  - Wave 1: freq ~0.02, amp ~30%, speed ~0.001, color #3D7EFF, opacity ~0.15
  - Wave 2: freq ~0.015, amp ~25%, speed ~0.0008, color #6BA3FF, opacity ~0.12
  - Wave 3: freq ~0.025, amp ~20%, speed ~0.0012, color #93C5FD, opacity ~0.10
- Each wave: iterate x from 0 to width, compute y = centerY + sin(x * freq + time * speed) * (height * amp), draw path with ctx.stroke()
- Add soft glow: set `ctx.shadowBlur = 15` and `ctx.shadowColor` matching wave color at 35% opacity (glow token)
- When reduced motion: render a single static frame (one drawWaves call, no rAF loop) OR a very slow opacity pulse via CSS
- `aria-hidden="true"` on canvas (decorative)
- IMPORTANT: Use `useCallback` for the draw function to prevent re-creating rAF on every render
