  <files>src/components/shared/MotionProvider.tsx, src/components/shared/ScrollReveal.tsx</files>
  <action>
Create `src/components/shared/` directory.

**MotionProvider.tsx** ("use client"):
- Import `LazyMotion`, `domAnimation` from `motion/react`
- Accept `{ children: React.ReactNode }` props
- Return `<LazyMotion features={domAnimation}>{children}</LazyMotion>`
- Use `domAnimation` NOT `domMax` (no layout animations or drag needed)

**ScrollReveal.tsx** ("use client"):
- Import `m` from `motion/react` and `useReducedMotion` from `motion/react`
- Props: `{ children: React.ReactNode; className?: string; stagger?: boolean }`
- When `stagger` is false (default): wrap children in a single `m.div` with:
  - `initial={{ opacity: 0, y: 40 }}` (40px slide up)
  - `whileInView={{ opacity: 1, y: 0 }}`
  - `viewport={{ once: true, amount: 0.2 }}` (20% threshold, play once)
  - `transition={{ duration: 0.6, ease: "easeOut" }}`
- When `stagger` is true: use parent/child variants pattern:
  - Container variant: `hidden: { opacity: 0 }`, `visible: { opacity: 1, transition: { staggerChildren: 0.12 } }`
  - Child variant exported as `staggerChildVariants`: `hidden: { opacity: 0, y: 40 }`, `visible: { opacity: 1, y: 0, transition: { duration: 0.5, ease: "easeOut" } }`
  - Container uses `m.div` with `variants`, `initial="hidden"`, `whileInView="visible"`, `viewport={{ once: true, amount: 0.2 }}`
  - Export `staggerChildVariants` so section components can apply them to individual children
- When `useReducedMotion()` returns true: set all `y` values to 0 and `duration` to 0 (instant visibility, no motion)
- IMPORTANT: Use `m.div` NOT `motion.div` -- LazyMotion requires the lightweight `m` proxy

Also export `staggerChildVariants` as a named export for use by section components that need child stagger.
